# X (Twitter) Clone

A simple frontend clone of X (Twitter) using **HTML** and **Tailwind CSS v4**.

## Features

- Clean X-like layout  
- Fully responsive  

## Tech Stack

- HTML5  
- Tailwind CSS v4

## Usage

1. Clone the repo  
2. Open `index.html` in your browser  

## Tailwind

Install Tailwind CLI:
https://tailwindcss.com/docs/installation/tailwind-cli
